# Predictive Analysis of Student Placement Success

## Project Overview

This project predicts whether a student will be placed based on academic and skill-related factors using Machine Learning.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Power BI

## Project Workflow

Raw Dataset
→ Data Cleaning
→ Exploratory Data Analysis (EDA)
→ Machine Learning Model
→ Prediction
→ Cleaned Dataset
→ Power BI Dashboard

## Machine Learning Model

- Logistic Regression

## Power BI Dashboards

- Placement Overview
- Student Performance Analysis
- Placement Insights
